This repo contans Google Earth Engine Java script and R code to create snowmelt day of year rasters from 2019-2025 for plant phenological study to match 2001-2018 data generated by  https://doi.org/10.3334/ORNLDAAC/1712. These data will be used to assess responses of plant phenology to snowmelt changes in the Rocky Mountains.
